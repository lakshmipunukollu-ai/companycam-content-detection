# Story 2: Project Management

## Description
As a contractor, I need to create and manage projects (job sites) so I can organize my photos by location.

## Acceptance Criteria
- CRUD operations for projects
- Projects have name, description, address, status
- Projects belong to a user (owner_id)
- List projects with filtering by status
- Only project owner can update/delete

## Technical Notes
- Projects are the top-level grouping for photos
- Reports are generated per project
